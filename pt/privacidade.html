O que a gente NÃO faz

  • Sem cookies. Sem trackers. Sem analytics.

Hospedagem

O site é hospedado na statichost.eu (Variable Object Assignment, Estocolmo/Suécia; servidor em Helsinque/Finlândia). A Statichost não armazena dados pessoais, endereços IP nem access logs.

Contato

Quando você envia um e-mail (hallo@zoebarossi.com ou sponsoring@zoebarossi.com), os dados enviados ficam armazenados para processar a solicitação. Base legal: Art. 6 par. 1 al. b do RGPD. Os dados são apagados depois que o assunto é resolvido, desde que não haja obrigação legal de guardá-los.

Links para redes sociais

Este site tem links para Instagram, Twitch, YouTube e eventualmente outros serviços. Só ao clicar num desses links é que dados são transmitidos aos respectivos provedores. Valem as políticas de privacidade de cada um deles.

Seus direitos

Você tem direito à informação, retificação, exclusão, limitação do tratamento, portabilidade dos dados e oposição. Para exercê-los, escreva para o e-mail acima.

Você também tem direito de apresentar reclamação a uma autoridade supervisora. No caso da Alemanha, é a autoridade de proteção de dados do respectivo estado (Bundesland).

Criptografia SSL

O site usa criptografia SSL por motivos de segurança. Dá para reconhecer pelo cadeado na barra de endereço do navegador.
